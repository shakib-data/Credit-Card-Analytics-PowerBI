💳 Credit Card Analytics Dashboard (Power BI)
📊 Project Overview

This Power BI project provides a comprehensive analysis of credit card transactions and customer data.
The goal of this report is to help a financial institution gain valuable insights into revenue trends, customer behavior, and overall performance.

The analysis is divided into two main interactive dashboards:

✨ Key Reports & Insights
🧾 1. Transaction Report

KPIs: Tracks Total Revenue, Total Interest, and Transaction Count.

Segmentation: Analysis by Card Type (Silver, Blue, Gold, Platinum) and Transaction Type.

Technology Usage: Revenue breakdown by Swipe vs Chip transactions.

👥 2. Customer Report

Demographics: Revenue analysis by Age Group, Marital Status, and Education Level.

Customer Insights: Breakdown by Job Role and Dependents.

Geographical Trends: Identifies Top 5 States contributing the highest revenue.
